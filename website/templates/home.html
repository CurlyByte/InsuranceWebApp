Welcome to Insurance Web App!


InsuranceWebApp Project:
This project was created based on the assignment of a Python course. As an extended project task, I programmed a web application that serves to create user accounts and admin account. Through Admin I can then create, add insurance for individual users and browse their data.
Programming Stack:
- The website was built using Flask - a Python web framework.
- SQLAlchemy with an SQLite database are used for data management.
- Development and coding were done in VS Code.
- Git is used for version control. - Bootstrap for making web more user-friendly.
Deployment Environment:
- The website is currently running on Linode (VPS) with Ubuntu as its OS.
- Nginx is configured as webserver and the Flask app is managed by the Gunicorn.
Security Measures
- Passwords are securely stored using hash functions.
- User roles -> admin and user have separated permission to ensure that users can't view and modify others accounts.
- Certbot is used for encrypting website with SSL/TLS certificate to ensure that web traffic is encrypted and secure.
